## Documento de Identidade Visual e Diretrizes de Marca

## 1. APRESENTAÇÃO DA MARCA

O **Festival da Graviola** é uma feira musical brasileira que celebra a cultura, a arte e a
música do Brasil, com inspiração no movimento Tropicália. A marca transmite energia,
autenticidade, alegria e um profundo respeito pelas raízes culturais brasileiras,
especialmente a cena musical baiana.

**Público-alvo:** Pessoas entre 18 e 35 anos que apreciam experiências autênticas,
música popular brasileira, arte visual e eventos culturais com propósito.

**Posicionamento:** Verão na Bahia, Hype-Vintage, Boêmio, Praiano, com uma
abordagem que mistura tradição e modernidade.

## 2. LOGO E MARCA

### Logo Principal

A logo da Graviola apresenta uma tipografia **arredondada e groovy** , estilo “bubble”,
em **amarelo vibrante (#F 2 B 705 )**. O design é acessível, memorável e transmite
movimento e energia.

**Variações de Logo:**

```
Versão horizontal (com espaço para texto complementar)
Versão vertical (compacta, para aplicações pequenas)
```

```
Versão monocromática (para aplicações em preto e branco)
```
**Proteção da Logo:**

```
Nunca alterar as proporções, cores ou tipografia da logo
Manter espaço em branco mínimo de 20 px ao redor da logo
Não sobrepor elementos visuais sobre a logo
Não usar em fundos que comprometam a legibilidade
```
### Selos e Patches

A marca utiliza selos decorativos em diferentes formatos (oval, hexagonal, retangular
com borda de costura) contendo a logo, o ano e elementos complementares como
frutas estilizadas. Esses elementos reforçam a identidade e são usados em peças de
comunicação.

## 5. ELEMENTOS GRÁFICOS DE APOIO

### Padrões

```
Padrão Tropical: Repetição de ícones de frutas (graviola, coco, banana, abacate)
em estilo pop-art, com cores da paleta principal
```

```
Padrão Minimalista: Pequenos ícones espaçados (guitarra, sol, folhas) para
```

```
Textura de Papel: Ruído sutil e grain para simular papel reciclado ou linho
Textura de Lambe-Lambe: Bordas rasgadas e irregulares, simulando colagem
manual
Textura Vintage: Film grain leve para fotos e composições
```
### Formas e Elementos Decorativos

```
Molduras Orgânicas: Formas irregulares, onduladas e naturais
Recortes de Colagem: Simulando cortes de revista e jornal
Fitas Adesivas Virtuais: Washi tape em cores da paleta
Selos e Patches: Formatos variados (oval, hexagonal, retangular com borda de
costura)
Ícones Tropicais: Guitarra, sol, folhas de palmeira, frutas estilizadas
```

## 8. PADRÕES E BOAS PRÁTICAS

### Hierarquia Visual

```
1. Nível 1 : Logo, título principal (Amarelo Graviola)
2. Nível 2 : Subtítulos, informações importantes (Verde Mata ou Vermelho Paixão)
3. Nível 3 : Corpo de texto, informações secundárias (Preto ou Cinza)
```
### Espaçamento

```
Manter espaço em branco mínimo de 20 px ao redor de elementos principais
Usar padding de 16 - 32 px entre seções
Alinhar elementos em grid de 8 px para coerência
```
### Proporções

```
Manter proporção 1 : 1 para ícones
Usar 3 : 4 para posts portrait
Usar 16 : 9 para apresentações
Usar 9 : 16 para stories
```
### Combinações de Cores Recomendadas

```
Amarelo + Verde: Combinação principal, energética
Amarelo + Vermelho: Destaque, urgência
Verde + Bege: Suave, sofisticado
Azul + Laranja: Alternativa moderna
```

```
Terracota + Ocre: Alternativa vintage
```

## 10. RECOMENDAÇÕES FINAIS

### Consistência

```
Usar sempre a paleta principal em 80 % das aplicações
Alternar paletas alternativas apenas em campanhas específicas
Manter a tipografia groovy para títulos em todas as peças
```
### Evolução Responsável

```
Não alterar a logo ou cores principais
Experimentar com padrões e elementos gráficos
Manter a essência Tropicália e brasileira
```
### Qualidade

```
Sempre usar imagens em alta resolução
Testar legibilidade em diferentes tamanhos
Revisar alinhamento e espaçamento
```
### Acessibilidade

```
Garantir contraste suficiente entre texto e fundo
Usar fontes legíveis em tamanhos pequenos
Descrever imagens em textos alternativos
```
